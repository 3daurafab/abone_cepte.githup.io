## GitHub Pages ile yayınlama

1. GitHub'da `kullaniciadiniz.github.io` isimli herkese açık bir depo oluşturun.
2. Bu ZIP içindeki dosyaları deponun kök dizinine yükleyin.
3. Depoda `Settings → Pages` bölümünü açın.
4. `Deploy from a branch` seçeneğini seçin.
5. Branch olarak `main`, klasör olarak `/ (root)` seçin.
6. `Save` düğmesine basın.
